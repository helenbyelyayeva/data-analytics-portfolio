# Suspicious Activity Monitoring Dashboard

## Overview
SQL and Tableau project focused on identifying suspicious transaction patterns and high-risk customer behavior.

## Business Problem
Financial institutions need to detect potentially suspicious transactions across customers, cards, merchant categories, and geographic locations.

## Objectives
- Detect high-risk MCC activity
- Identify suspicious clients and cards
- Analyze transaction behavior by age group
- Detect cards active across multiple states
- Highlight top suspicious customers

## Tools Used
- SQL
- Tableau
- Excel

## Dashboard Features
- Suspicious transaction amount
- Suspicious transaction count
- Suspicious clients
- Suspicious cards
- Clients with more than 5 cards
- High-risk MCC distribution
- Geographic analysis
- Top suspicious clients
- Multi-state card activity
